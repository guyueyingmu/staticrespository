main.py 中配置说明：

19行：数据库账户相关信息，修改为你自己的信息

49行：sleep(1)代表1s发送一条， 可以控制频率 （启动控制的话去掉前面的注释符号#）



fun.py 文件

17、18行： 设置发布豆油账户信息  如果账户被加入黑名单 可以更换这个账户 一般一个就可以了



run.bat

直接双击运行 如果有验证码出现，查看同目录下的v.jpg验证码图片，然后输入验证码即可 


keywords.txt 

为随机内容文件， 有效防止验证码的出现， 理论上越大出现验证码的几率越小 



提示：

频率过快可能会导致发豆油的过程出现验证码， 一般会偶尔出现  验证后还可继续连续的发布豆油
